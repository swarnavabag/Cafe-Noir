# ☕ Café Noir

A modern, elegant, and fully responsive café website designed to deliver a premium digital experience for visitors.
Café Noir combines aesthetic UI design, smooth interactions, and clean layouts to reflect the ambiance of a luxury café.

## 📖 Project Overview

Café Noir is a static frontend website built to simulate the digital presence of a premium café brand.

The website focuses on:

* Strong visual identity
* Elegant typography
* Clean layout composition
* Smooth hover interactions
* Responsive design for multiple screen sizes

The design emphasizes whitespace, warm tones, and luxury-inspired styling to create a refined user experience.

## 📱 Responsive Design

This website is optimized for:

* **Desktop** → 1024px and above
* **Tablet** → 768px to 1024px
* **Mobile** → Below 768px

Responsive techniques used:

* Media queries
* Flexible layouts
* Relative sizing
* Adaptive typography
* Responsive spacing

## 🚀 Running Locally

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/cafe-noir.git
```

Move into the project directory:

```bash
cd cafe-noir
```

Open:

```bash
index.html
```

Or run using Live Server in VS Code.
